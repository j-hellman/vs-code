
### Shortcuts

* **Copy line:** 
  - shift + alt + up/down  (WIN)
  - shift + option + up/down  (MAC)

* **Join line:**
  - shift + alt + J  (conf. pessoal. P/ mudar: File>preferences>keyboards shortcuts>type:editor.action.joinlines)

* **Code formatting:**
  - shift + alt + F

* **Rename Symbol**
  - Select the symbol and them press F2

* **Navigate to a specific line:**
  - ctrl + G (WIN / MAC)

* **Column selection:**
  - shift + alt + mouse selection

* **Select word:**
  - ctrl + d (WIN)
  - command + d (MAC)

* **Duplicate cursor:**
  - ctrl + alt + up/down

* **Open terminal:**
  - ctrl + shift + ` (WIN)

* **Openning/Closing side bar:** 
  - ctrl + b (WIN)
  - command + b (MAC)

* **Find:**
  - ctrl + F (WIN)

* **Delete previous word:**
  - ctrl + backspace (WIN)
  - command + delete (MAC)

* **Delete line:**
  - ctrl + X
